# Movie Review Console App

**사용자가 영화를 등록하고 리뷰(별점과 감상평)를 작성하면, 영화별 평균 평점을 계산해주는 콘솔 기반 프로그램입니다.**

---

## 주요 기능

### 1. 입력
1. **영화 등록**
    - 사용자가 **영화 제목**과 **장르**를 입력하여 새로운 영화를 등록합니다.
    - 중복된 영화는 등록할 수 없습니다.
2. **리뷰 작성**
    - 등록된 영화 목록에서 하나를 선택하여 리뷰를 작성합니다.
    - 평점은 **0.5** 단위로 입력하며, 0.0~5.0 범위만 허용됩니다.
    - 감상평이 공백일 경우 등록되지 않습니다.
3. **리뷰 조회**
    - 등록된 모든 영화의 리뷰를 확인할 수 있습니다.
    - 각 영화의 평균 평점을 함께 출력합니다.
4. **종료**
    - 프로그램이 종료됩니다.

### 2. 평균 평점 계산
- 입력받은 평점을 기준으로 평균 평점을 계산합니다.
- 리뷰가 없는 영화의 경우 0.0으로 표시됩니다.

### 3. 출력
- **평점 순 정렬**
    - 영화 목록은 **평균 평점**이 높은 순으로 정렬되어 출력됩니다.
- **리뷰 출력**
    - 각 영화의 제목, 장르, 평균 평점, 리뷰 내용을 함께 표시합니다.

### 4. 예외처리
- 예외 발생 시 IllegalArgumentException을 발생시킨 후 [ERROR]로 시작하는 에러 메세지를 출력합니다.
- 입력값이 잘못된 경우 오류 메세지를 출력한 뒤 해당 입력 단계부터 재입력을 받습니다.
- **예외 상황 목록**
|구분|조건|예외 메시지 예시|
|---|---|---|

## 테스트

### 1. 정상 동작 테스트
|테스트 항목|검증 내용|
|----|---|
|영화 등록|제목과 장르 입력 시 영화가 등록되는지 검증|
|리뷰 작성|영화를 선택하고, 평점을 입력하고, 내용을 작성하면 리뷰가 등록되는지 검증|
|리뷰 조회|영화 목록과 리뷰가 제대로 출력되는지 검증|

### 2. 예외 테스트
|테스트 항목|기대 결과|
|0.0~5.0 범위를 벗어난 평점 입력의 경우| IllegalArgumentException 발생|
|평점을 음수로 입력한 경우| IllegalArgumentException 발생|
|영화 제목을 공백으로 입력한 경우| IllegalArgumentException 발생|
|기존 등록한 영화의 제목과 중복된 경우| IllegalArgumentException 발생|
|존재하지 않는 영화의 ID를 입력한 경우| IllegalArgumentException 발생|
